# The Last One
The World is industrial and dying, no green left anywhere. Somehow one tree survived. You and other players get powers from god — no explanation — to protect it. Enemies come for it. You defend it together.

## Game Mechanics
The fun thing about this game, is you use your voice to play! There are trigger words that are picked up by our system that trigger different events.
The words are: <br>

Shapeshift: = shapeshift/morph <br>
Flight: flight, fly, ascend, rise, up <br>
4th Wall: kill <br>
Right left and up keys to move player
Shoot = Shoot
